Botones

inputs

Para usar los inputs basta con la clase input como contenedor principal.

Si es tipo texto se le añade como segunda clase input-text 


            <div class="input input-text">
                <input type="text">
            </div>

iconos

Para añadirle un icono solo debe colocar un div con la etiqueta icon dentro del contenedor 


            <div class="input input-text">
            <div class="icon">X</div>
                <input type="text">
            </div>
